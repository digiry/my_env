## Prerequisite

### ripgrep

fzf-preview 에서 ProjectGrep 에서 사용하는 명령

**rg command**

```bash
brew install ripgrep
```

### bat

syntax highlighting

```bash
brew install bat
```

### deno

```bash
brew install deno
```
